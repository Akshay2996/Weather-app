## Run Locally

Clone the project

```bash
  git clone https://github.com/Akshay2996/Weather-app.git
```

Go to the project directory & Install dependencies

```bash
  cd Weather-app
  npm install
```

Start the server

```bash
  npm run start
```

**Note:** Get your API keys from the API website and store it in .env file. Sample given.
